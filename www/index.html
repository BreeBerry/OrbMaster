⚔️ How to Play
OrbMaster is a code-breaking RPG. Each boss guards a secret sequence of colored orbs — crack the code before you run out of turns.
The Core Loop
1. Choose your boss from the Campaign screen. Bosses unlock in order as you defeat each one.
2. Set your loadout — equip up to 4 ability orbs from your stash before entering battle.
3. Study the Orb Pool — each boss has a fixed set of colors their code is drawn from, shown in the pre-battle loadout.
4. Submit a guess — place colored orbs into the slots and tap Submit.
5. Read the feedback pegs — they tell you how close your guess was.
6. Use orb abilities strategically to gain hints, extra turns, or eliminate colors.
7. Crack the code within the turn limit to earn orb fragments.
8. Spend fragments in the Orb Forge to craft powerful ability orbs.
Win & Loss
Win: Every orb in your guess is the correct color in the correct position.
Lose: Run out of turns without cracking the code.
Feedback Pegs
Red Peg — correct color, correct position
White Peg — correct color, wrong position
No Peg — color not in the code at all
Pegs show only the total count of reds and whites — not which slot each peg belongs to.
Turn Limit
Each match starts with 10 turns. Orb abilities can add extra turns. Some boss specials can remove them.
The faster you crack the code, the more fragments you earn.
🔒 Orb Locking
Double-tap a guess slot to lock that color to that position. Locked orbs carry forward into every future guess automatically. A gold glow marks locked slots. Double-tap again to unlock.
✕ Color Elimination
Triple-tap a color in the picker to eliminate it — graying it out when you're confident it's not in the sequence. Double-tap a grayed color to restore it.
Locked and eliminated colors are automatically filtered out by orb abilities — so hints and reveals focus on orbs you haven't figured out yet.
MindBreaker Altar
The MindBreaker Altar is your guess console — a stone altar with orb slots carved into its face. It determines how many orb slots you can use per guess.
Campaign Mode
In Campaign, each boss sets the code size automatically based on their difficulty. Bosses 1–7 use 4-orb codes, Bosses 8–9 use 5-orb codes, and Boss 10 uses a 6-orb code. The MindBreaker rank does not affect Campaign matches.
Multiplayer Mode
In Play Online (Multiplayer), the host chooses the code size — 4, 5, or 6 orbs. Both players must use the same code size. Your MindBreaker rank determines which sizes you can access.
4 Slots — Available from the start.
5 Slots — Requires Rank 5 upgrade from the Orb Forge.
6 Slots — Requires Rank 6 upgrade from the Orb Forge.
If your opponent's chosen code size exceeds your rank, you'll see a warning. Ask them to lower the size or upgrade your altar first.
Upgrade Costs
Rank 5 (5 slots): 150 Orange + 150 Blue + 100 Green + 100 Red fragments. Requires beating Boss 5.
Rank 6 (6 slots): 200 Purple + 150 White + 150 Black + 100 Teal fragments. Requires beating Boss 7.
Deeper codes earn bigger fragment rewards — ×1.3 for 5 slots, ×1.7 for 6 slots.
🔮 Mind Orbs
There are 12 Mind Orbs. Each orb is both a puzzle color (appearing in boss codes) and a forgeable ability item you can equip. New colors enter the boss pool as you progress through the campaign — each boss beyond the third introduces one new orb into the mix.

👹 Boss Profiles
Each boss guards a secret orb code. Defeat them in order to progress the campaign and unlock new colors, ranks, and abilities.
Fragment System
Orb Fragments are the currency of OrbMaster. You earn them by winning matches and spend them in the Orb Forge to craft powerful ability orbs.
Earning Fragments
Each match rewards 4 batches of fragments when you win. The type and amount depend on how quickly you solved the code and the boss's loot table.
Base reward per orb: 50 fragments
Turn penalty: −5 fragments per orb for every turn used after Turn 1
Formula: (50 − ((turns_used − 2) × 5)) × code_size × multiplier
Solving in Turn 1 or 2 gives the maximum reward. Using orb abilities that add turns does not reduce your reward.
Reward Multipliers
5-orb code: ×1.3 bonus to all fragments earned
6-orb code: ×1.7 bonus to all fragments earned
Pink Orb equipped: ×1.75 total multiplier
Teal Orb equipped: ×2.25 total multiplier (does not stack with Pink)
Boss 7+: +20% bonus fragments (late-game economy boost)
Loot Rarity
Each fragment batch is rolled against the boss's loot table to determine its rarity tier — Common, Uncommon, or Rare. Higher rarity means rarer orb types appear in the reward.
Silver Orb (+15% Uncommon) and Gold Orb (+20% Rare) passively shift the loot table in your favor when equipped.
Forging Orbs
Visit the Orb Forge to spend fragments and craft ability orbs. Each orb has its own fragment cost — any orb can be forged once you have enough fragments, regardless of campaign progress.
🗺️ Progression & Unlocks
OrbMaster has a full Campaign of 10 bosses. Defeat each boss to unlock the next. No fragment requirements — just beat the previous boss to advance.
Boss Orb Pool Progression
Each boss draws their secret code from a fixed orb pool. The pool grows with each boss, adding one new color per boss from Boss 4 onward.
Bosses 1–3: Red, Blue, Yellow, Green, Orange (5 orbs)
Boss 4: + Silver (6 orbs)
Boss 5: + Gold (7 orbs)
Boss 6: + Purple (8 orbs)
Boss 7: + Black (9 orbs)
Boss 8: + White (10 orbs) — 5-orb code
Boss 9: + Pink (11 orbs) — 5-orb code
Boss 10: + Teal (12 orbs) — 6-orb code
The full orb pool for each boss is shown in the pre-battle loadout screen. You'll only see those colors in the battle picker.
MindBreaker Altar (Multiplayer)
The altar rank only affects Multiplayer code size selection. Campaign code sizes are fixed by each boss.
Rank 5 (5 slots): Purchase at Orb Forge after Boss 5.
Rank 6 (6 slots): Purchase at Orb Forge after Boss 7.
Settings
Duplicate Colors: Toggle in Settings to allow or disallow repeated colors in boss codes. On by default — turning it off makes codes harder to guess.
Build Archetypes
🧠 Solver Build: Black + Red + Green — maximum information per turn.
💰 Economy Build: Pink + Gold + Silver — maximize fragment rewards.
🛡️ Control Build: Yellow + Blue + Purple — board manipulation and extra turns.
💪 Endurance Build: Orange + Purple + White — stack extra turns for harder bosses.
